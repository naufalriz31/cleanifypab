# CLEANIFY

CLEANIFY adalah aplikasi mobile berbasis **Kotlin** dan **Jetpack Compose** yang digunakan untuk
memonitor dan mencatat aktivitas kebersihan ruangan secara digital. Aplikasi ini membantu
menggantikan metode manual seperti checklist kertas atau laporan lisan, sehingga proses
monitoring kebersihan menjadi lebih rapi, terstruktur, dan terdokumentasi.

Aplikasi ini memungkinkan petugas kebersihan memperbarui status dan checklist ruangan
secara cepat melalui QR Code, sementara admin atau supervisor dapat memantau kondisi
ruangan serta riwayat kebersihan secara real-time.

---

## Fitur Utama

- Scan QR Code ruangan
- Checklist kebersihan ruangan
- Update status ruangan (Bersih / Perlu Dibersihkan / Belum Dibersihkan)
- Manajemen data ruangan oleh Admin
- Penugasan task kepada petugas kebersihan
- Monitoring kondisi ruangan secara real-time
- Riwayat kebersihan harian
- Antarmuka sederhana dan mudah digunakan

---

## Teknologi yang Digunakan

- **Kotlin**
- **Jetpack Compose**
- **MVVM Architecture**
- **Firebase Firestore**
- **Firebase Authentication**
- **CameraX** (QR Code Scanner)
- **Android Studio**

---

## Arsitektur Aplikasi

Aplikasi CLEANIFY menerapkan arsitektur **MVVM (Model–View–ViewModel)** untuk memisahkan
logika bisnis, pengelolaan data, dan tampilan antarmuka pengguna. Pendekatan ini membuat
kode lebih terstruktur, mudah dipelihara, dan scalable.

---

## Cara Menjalankan Aplikasi

1. Clone repository ini ke komputer Anda
2. Buka project menggunakan **Android Studio**
3. Lakukan **Gradle Sync**
4. Jalankan aplikasi pada **emulator** atau **perangkat Android**
